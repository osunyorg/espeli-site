Comment faire pour enregistrer votre bateau ?

header_text: >- header_cta: display: false label: >- target: "" external: false contents_reading_time: seconds: 70 text: >- 1 minute contents: - kind: block template: chapter title: >- slug: >- ranks: base: 2 top: active: false data: layout: no_background text: >-

La démarche d'enregistrement est différente pour le maritime (navires en mer) et pour le fluvial (bateaux). Cette page concerne uniquement l'enregistrement en eaux intérieures (fluvial). Pour le maritime, rendez-vous sur le portail démarches plaisance (lien externe).

notes: >- alt: >- credit: >- - kind: block template: title title: >- Quelques définitions slug: >- quelques-definitions ranks: base: 2 self: 2 top: active: true title: value: >- Quelques définitions heading: 2 data: layout: classic - kind: block template: definitions title: >- slug: >- ranks: base: 3 top: active: false data: description: >- elements: - title: >- Devise du bateau description: >-

La devise est le nom du bateau. Il faut au moins 3 caractères dans le champ du CERFA.

- title: >- Eaux intérieures description: >-

Les fleuves, rivières, lacs et barrages.

- title: >- Nolisage description: >-

Location de bateau. Le noliseur est un loueur de bateau.

- title: >- Propriétaire description: >-

Personne physique ou morale (banque, noliseur...) qui possède le bateau.

- title: >- Titulaire description: >-

Personne physique ou morale référencée sur la carte d'enregistrement.
